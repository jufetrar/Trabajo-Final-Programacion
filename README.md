
Felipe Triana Proyecto Final-Programación 

FitTrack Pro
Trabajo final de Programación — plataforma web o App para unir gimnasio y running en un solo lugar.

¿Por qué este tema?

Elegi este proyecto porque queríamos hacer algo que realmente le sirviera a alguien más allá del salón de clase. Mucha gente quiere empezar a hacer ejercicio pero no sabe por dónde arrancar, o empieza y no lleva ningún control de lo que hace, entonces no ve su progreso y termina dejándolo. FitTrack Pro busca resolver eso: una herramienta simple donde cualquier persona, sin experiencia previa, pueda registrar sus entrenamientos de gimnasio y sus carreras, ver su progreso semana a semana y mantenerse motivada con logros y rachas.

¿Qué hace la plataforma?

Registro e inicio de sesión de usuarios, con validación de formularios y medidor de fuerza de contraseña.
Dashboard personal con resumen semanal: entrenamientos realizados, calorías quemadas, distancia recorrida y racha de días activos.
Gym Tracker: registro de rutinas y ejercicios, con incremento de peso configurable (sobrecarga progresiva).
Running: registro de carreras con distancia y duración, cálculo automático de ritmo.
Logros y estadísticas para ver la evolución del usuario en el tiempo.
Configuración: cambio de idioma (español/inglés) y modo oscuro/claro.
Todo se guarda en el navegador (localStorage), separado por usuario, así que cada persona ve solo sus propios datos.

Tecnologías usadas
HTML5
CSS3 (variables CSS, diseño responsive)
JavaScript (vanilla, sin frameworks ni librerías externas de lógica)
Font Awesome (íconos)
localStorage del navegador como almacenamiento de datos (no hay backend ni base de datos real)

Cómo probarlo

No necesita instalación ni servidor: es HTML/CSS/JS puro.

Clona el repositorio o descárgalo como ZIP.
Abre el archivo PROGRAMACION TRABAJO FINAL/index.html en tu navegador.
Regístrate con cualquier correo y contraseña (mínimo 8 caracteres).
Explora el dashboard: agrega entrenamientos de gimnasio o carreras y observa cómo cambian las estadísticas.
